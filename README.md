# gameProject : 짱구런
### 초코비를 먹으며 방사능 브로콜리를 피해 점수를 획득하는 게임입니다. (ex: 쿠키런)
### 사용자별 랭킹 기록을 위해 로그인 기능 구현

### ◆ 개발 환경
* Eclipse
* Database : MySQL

### ◆ 학습 목표
* Java 프로그래밍 언어 이해
* 데이터베이스 구성과 정보 저장의 이해
* 객체지향 프로그래밍 개념 이해와 적용방법

### ◆ 담당 업무
* 게임 캐릭터와 아이템 디자인
* UI 디자인 구성
* 데이터베이스 구성요소 설계
